## 🚨 Common Issues & Solutions

### Issue: Port 5173 Already in Use
**Solution:** Vite automatically tries next ports (5174, 5175...)

### Issue: Blank Page
**Solution:** Check browser console for errors, refresh page, check that npm install completed

### Issue: Styles Not Loading
**Solution:** Tailwind CSS requires restart. Stop server and `npm run dev`

### Issue: Dark Mode Not Working
**Solution:** Clear LocalStorage: `localStorage.clear()` in browser console

### Issue: Draft Not Saving
**Solution:** Check if LocalStorage is enabled in browser settings
